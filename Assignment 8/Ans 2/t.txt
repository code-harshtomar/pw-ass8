Absolute =
-Syntax
{
    position:absolute;
}
-with relative positioning, an element can be moved up,down ,left or right from its normal position.
-It will not break the normal document flow to position the element on the page.
-The element will leave the space at its original position & other content will note be adjust to fit into any gap left by the element.


Relative =
-Syntax
{
    position:absolute;
}
-with absolute positioning an element is posiioned relative to its nearest positioned ancestor.
-It will breal the document flow to position the elemnet on the page.
-The element will not leave any space in its original position.
